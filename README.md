基于 YOLOv8 的垃圾分类检测系统
同济大学课程大作业 | 校园场景测试

项目说明
本项目为课程大作业实践，基于 YOLOv8 实现生活垃圾目标检测，
并在同济大学校园实拍场景下完成测试验证。
